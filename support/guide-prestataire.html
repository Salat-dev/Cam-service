Étape 1

Créer votre compte prestataire

La première étape pour rejoindre la communauté CamServices.

Rendez-vous sur la page d'inscription

Cliquez sur "Devenir prestataire" depuis la page d'accueil ou directement ici.

Remplissez le formulaire en 4 étapes

  1. Compte : Nom complet, nom d'utilisateur, email, mot de passe, téléphone, photo de profil
  2. Adresse : Ville et quartier pour que les clients vous trouvent facilement
  3. Profil Pro : Situation actuelle, domaines d'expertise, formation, activités parallèles
  4. Vérification (optionnelle) : Pièce d'identité pour obtenir le badge "Vérifié"

Conseil : Ajoutez une photo de profil professionnelle. Les profils avec photo reçoivent 3 fois plus de demandes.

Étape 2

Découvrir votre tableau de bord

Votre espace de gestion centralisé.

Vue d'ensemble

La page d'accueil du dashboard affiche vos statistiques clés : revenus, commandes, vues du profil. C'est votre tableau de bord principal.

Navigation

La barre latérale gauche vous permet d'accéder à toutes les sections :

  • Mes services : gérez vos offres
  • Commandes : suivez les demandes des clients
  • Grille tarifaire : définissez vos tarifs détaillés
  • Portfolio : montrez vos réalisations
  • Activités : certifications et formations
  • Mon profil : modifiez vos informations
  • Statistiques : analysez vos performances
Étape 3

Créer votre premier service

C'est le cœur de votre activité sur CamServices.

Cliquez sur "Nouveau service"

Depuis la page Mes services, cliquez sur le bouton "Nouveau service".

Remplissez les informations

  1. Titre : soyez clair et descriptif. Ex: "Plombier professionnel - Intervention rapide"
  2. Catégorie : choisissez la catégorie qui correspond le mieux
  3. Type de prix : fixe, horaire, journalier ou par projet
  4. Prix : indiquez un prix compétitif en FCFA
  5. Description : détaillez ce que vous proposez, votre expérience, ce qui est inclus
  6. Image : ajoutez une photo de votre travail (URL ou upload)
  7. Jours de travail : sélectionnez vos jours de disponibilité
  8. Horaires : définissez vos heures de début et de fin

Conseil : Un service avec une image et une description détaillée reçoit 5 fois plus de clics.

Étape 4

Ajouter des réalisations à votre portfolio

Montrez votre savoir-faire avec des photos de vos meilleurs travaux.

Pourquoi un portfolio ?

Les clients veulent voir votre travail avant de commander. Un portfolio bien rempli augmente votre crédibilité et rassure les clients potentiels.

Comment ajouter une réalisation

  1. Allez dans Portfolio depuis le menu latéral
  2. Cliquez sur "Ajouter une réalisation"
  3. Donnez un titre (ex: "Maquillage mariage traditionnel")
  4. Ajoutez une catégorie et une description
  5. Ajoutez l'image (URL ou upload)
  6. Cochez "Mettre en vedette" pour les meilleures réalisations

Bonne pratique : Ajoutez au moins 5 réalisations variées pour montrer l'étendue de vos compétences.

Étape 5

Définir votre grille tarifaire

Des tarifs clairs pour attirer plus de clients.

Pourquoi une grille tarifaire ?

Une grille tarifaire détaillée permet aux clients de savoir exactement combien ils vont payer. Cela élimine les surprises et augmente la confiance.

Types de tarification

  • Forfait : prix fixe pour une prestation complète
  • Horaire : prix par heure travaillée
  • Demi-journée / Journée : idéal pour les prestations longues
  • Par personne : pour les services dont le prix dépend du nombre
Étape 6

Gérer vos commandes

Suivez et traitez les demandes de vos clients.

Cycle de vie d'une commande

  1. En attente : un client a passé commande
  2. Confirmée : vous acceptez la commande
  3. En cours : vous avez commencé la prestation
  4. Terminée : la prestation est livrée

Actions disponibles

  • Confirmer : acceptez la commande
  • Annuler : refusez la commande
  • Démarrer : indiquez que le travail a commencé
  • Terminer : marquez la prestation comme livrée
  • Corbeille : supprimez une commande (restaurable)

Attention : Les informations du client (nom, téléphone, adresse) sont visibles dans le détail de la commande. Contactez le client via WhatsApp ou téléphone.

Étape 7

Optimiser votre profil

Un profil complet attire plus de clients.

Les éléments clés d'un bon profil

  • Photo de profil : claire et professionnelle
  • Nom complet : votre vrai nom ou nom commercial
  • Bio : parlez de vous, de votre expérience
  • Expertise : listez vos domaines de compétence
  • Formation : vos diplômes et certifications
  • Vérification : obtenez le badge "Vérifié"

Statistique : Les profils vérifiés reçoivent 2 fois plus de commandes que les profils non vérifiés.

Conseils

Conseils pour réussir sur CamServices

Les bonnes pratiques qui font la différence.

Soyez réactif

Répondez rapidement aux demandes des clients. Un temps de réponse court augmente votre taux de conversion.

Soignez vos photos

Des photos de qualité de vos réalisations valent mieux qu'un long discours. Investissez dans de bonnes photos.

Fixez des prix justes

Étudiez la concurrence et proposez des prix compétitifs. Vous pourrez les ajuster avec l'expérience.

Demandez des avis

Après chaque prestation réussie, encouragez vos clients à laisser un avis positif. Les avis rassurent les futurs clients.

Mettez à jour régulièrement

Ajoutez de nouvelles réalisations, ajustez vos tarifs, modifiez vos disponibilités. Un profil actif est mieux référencé.

Rappel : CamServices est gratuit pour les clients. Vous ne payez rien pour recevoir des commandes. Votre succès est notre priorité.
